## 🐾 **Pet Shop - Pet Store Website**

This is a pet store website built with **Next.js**, **TypeScript**, and **MySQL**. The website provides features for product management, shopping carts, and customer checkout.

---

### 🚀 **Getting Started**

First, install the required packages:
```bash
npm install
# or
yarn install
```

Then, start the development server:
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

---

### 📦 **Key Features**

- 🐶 Manage pet product listings  
- 🛒 Add to cart and checkout  
- 🔍 Search and filter products  
- 👤 User account management  
- 💬 Support chat and feedback

---

### 🛠 **Environment Configuration**

Create a `.env` file in the root directory and add the MySQL connection information:
```bash
DATABASE_URL="mysql://username:password@localhost:3306/petshop"
```
